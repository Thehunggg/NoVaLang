1. Update the approved Shared Source Of Truth.
2. Generate.
3. Validate.
4. Sync.
Never hand-edit generated JSON.

Lesson Format 2.0:
This file is the Single Source of Truth for the opt-in `lessonFormat: five_cards`
format.
Reference implementation: ja-daily_life-m01-u1-l1

Các trường source đã tìm thấy nguyên văn:
lessonFormat / targetLanguage / targetLocale / cultureContext /
approvedCharacterNamePool / displayText / canonicalText / audioText

Quy tắc của ba trường:
Store `displayText`, `canonicalText`, and `audioText` separately: UI renders
`displayText`, the answer checker uses `canonicalText`, and TTS uses `audioText`.

Cấu trúc bài five_cards:
Every `five_cards` lesson has exactly these five overview cards, in this order,
with each card opening its own page:
1. Introduction
2. Vocabulary cards
3. Short dialogue
4. Grammar
5. Exercises

- Explicit `lessonFormat: five_cards` opt-in and approved target language/locale.
- Approved source content, localization, character metadata, and reading policy.
- `targetLanguage` and `targetLocale` exist; when a lesson uses characters,
  `cultureContext` exists; when dialogue uses a speaker,
  `approvedCharacterNamePool` exists.
- Exactly five cards in the frozen order and the approved 14-question map.
- Free 1–10 and Plus 11–14 access plans; Q15 remains deferred.
- Stable IDs, local persistence/resume/wrong-answer behavior, and completion
  rules are represented in source and validation.
- Source → generate → validate → sync is available; no generated JSON is edited
  manually.
