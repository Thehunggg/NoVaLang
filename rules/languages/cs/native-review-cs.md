# cs — Native speaker review checklist

For a native Czech (cs-CZ) speaker. Simple tick-box items. Pure Czech language
facts the project owner (not a Czech speaker) cannot verify. None block the
current VALIDATED_NOT_YET_PROVEN status; they refine confidence.

No front-matter (native-review file, exempt).

---

## [N-cs-01] Case endings (7 cases × gender × animacy)

Confirm a compact declension paradigm (including vocative and consonant
alternations: kniha→knize, ruka→ruce) for the first lesson's nouns.

- [ ] Declension tables match standard reference.

## [N-cs-02] Masculine animacy

The animate/inanimate split (pán→pány / hrad→hrady) changes accusative + plural.
Confirm example sentences use the right form.

- [ ] Animacy agreement correct.

## [N-cs-03] Verb aspect pairs

Which perfective matches which imperfective (dělat/udělat, psát/napsat, brát/vzít)
is largely lexical. Confirm the first lessons' aspect pairs.

- [ ] Aspect pairs correct.

## [N-cs-04] Clitic second position

Confirm clitic ordering and second-position placement (Včera jsem se ti smál) in
example sentences.

- [ ] Clitic order + position correct.

## [N-cs-05] Consonant clusters + ř

Confirm pronunciation of dense clusters and ř in lesson vocabulary; ř is hard
even for learners.

- [ ] Clusters + ř correct in audio.

## [N-cs-06] spisovná vs obecná boundary

Confirm example sentences are consistent spisovná and any obecná annotations are
labeled, not mixed.

- [ ] Consistent spisovná; obecná labeled if present.

## [N-cs-07] Naturalness / register of translations

Per ADR-016: confirm example sentences sound natural (not literal), aspect + case
idiomatic, ty/vy appropriate.

- [ ] Sentences natural; aspect + case idiomatic.
