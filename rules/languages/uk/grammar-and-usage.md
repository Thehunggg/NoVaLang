# Ukrainian Grammar & Usage — Ngữ pháp tiếng Ukraina

Phần KHÓ NHẤT (chữ→âm thì dễ). Ukraina giữ hình thái Slavic phong phú.

## Không có mạo từ

Như mọi Slavic — xác định/không xác định qua trật tự + ngữ cảnh.

## 3 giống + animacy

masculine / feminine / neuter + tính **động vật** (ảnh hưởng accusative). Đoán
qua đuôi (phụ âm→masc, -а/-я→fem, -о/-е→neut), nhiều biệt lệ → lexical.

## 7 CÁCH — có VOCATIVE (đặc trưng Ukraina)

| Cách | Vai |
|---|---|
| Nominative (називний) | chủ ngữ |
| Genitive (родовий) | sở hữu/của, phủ định |
| Dative (давальний) | tân ngữ gián tiếp |
| Accusative (знахідний) | tân ngữ trực tiếp |
| Instrumental (орудний) | công cụ/'bằng', 'với' |
| Locative (місцевий) | nơi chốn (luôn có giới từ) |
| **Vocative (кличний)** | **gọi trực tiếp** |

Ukraina **giữ vocative sống**: Петре! Мамо! Друже! пане! — KHÁC Nga (đã mất).
Đây là điểm dạy đặc trưng. Danh từ + tính từ + đại từ đều chia; biến âm gốc
(стіл→стола, ніч→ночі) cần native.

## Động từ

- **Thể (aspect)**: доконаний/недоконаний (perfective/imperfective) — trục cốt lõi.
- Hiện tại: chia ngôi+số (2 nhóm: читаю/читаєш/читає; говорю/говориш).
- **Quá khứ**: chia **giống+số**, KHÔNG theo ngôi (читав m / читала f / читало n /
  читали pl) — đặc trưng Slavic.
- **Tương lai** imperfective: phân tích (буду читати) HOẶC **tổng hợp**
  (читатиму — đặc trưng Ukraina); perfective: прочитаю.
- Bất quy tắc: бути, дати, їсти, хотіти — lexical.

## Trật tự từ

SVO nền, **tự do cao** (cách đánh dấu vai). Giới từ + cách cụ thể (в/на +
locative/accusative).
