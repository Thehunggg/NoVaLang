## Đặc trưng tiếng Hungary nổi bật

- **Chắp dính** (agglutinative) — hậu tố xếp chồng (házaimban).
- **Hoà âm nguyên âm** — hậu tố đổi dạng theo gốc (-ban/-ben/-höz).
- **~18 hậu tố cách** + **chia định/bất định** (látom/látok).
- **s = [ʃ], sz = [s]** — ngược tiếng Anh (bẫy lớn).
- **á é í ó ö ő ú ü ű** chữ riêng — ngắn/dài đổi nghĩa (öt/őt); ö/ő ü/ű đặc trưng.
- **Trọng âm luôn âm tiết đầu**; **không giống ngữ pháp**; **pro-drop**;
  trật tự topic-focus.
- **te/ön** đối lập T-V (KHÁC scandinavia du-phổ-quát).
