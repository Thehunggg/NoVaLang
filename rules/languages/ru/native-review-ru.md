# ru — Native speaker review checklist

For a native Russian speaker. Simple tick-box items. Pure Russian language
facts the project owner (not a Russian speaker) cannot verify. None block the
current VALIDATED_NOT_YET_PROVEN status; they refine confidence.

No front-matter (native-review file, exempt).

---

## [N-ru-01] Case ending tables (6 cases × 3 genders × number)

The full declension paradigm (noun/adjective/pronoun endings by gender × case ×
number × declension class) is morphological lexical data, not yet compiled.
Confirm a compact reference paradigm before case-agreement exercises ship.

- [ ] Declension tables match standard reference (e.g. Zaliznyak).

## [N-ru-02] Verb aspect pairs

Which perfective matches which imperfective (делать/сделать, писать/написать,
брать/взять) is largely lexical. Confirm the first lessons' aspect pairs.

- [ ] Aspect pairs correct for lesson vocabulary.

## [N-ru-03] Noun gender + stress placement

Gender is mostly predictable from ending but has exceptions (папа m, путь m).
Stress is free and mobile (рука́ → ру́ки) and unmarked. Confirm gender + stress
mark for the first lesson's nouns.

- [ ] Gender + stress correct for lesson-1 nouns.

## [N-ru-04] Numeral–case agreement in examples

Confirm example sentences apply the correct case after numerals (один стол / два
стола / пять столов), including compound numerals (двадцать один, тридцать два).

- [ ] Numeral case correct in examples.

## [N-ru-05] ты/вы appropriateness

Confirm ты/вы choice fits each situation and that no dialogue forces the wrong
register (e.g. addressing a stranger with ты).

- [ ] ты/вы appropriate; name+patronymic used where expected.

## [N-ru-06] Stress marks (ударение) accuracy

If reading-aid stress marks are shown, confirm they sit on the correct syllable
for each word form (mobile stress changes across cases).

- [ ] Stress marks on correct syllable.

## [N-ru-07] Naturalness / register of translations

Per ADR-016: confirm example sentences sound natural to a native (not literal),
aspect choice fits context, and word order matches information focus.

- [ ] Sentences natural; aspect + word order idiomatic.
