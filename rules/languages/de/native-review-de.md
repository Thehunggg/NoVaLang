# de — Native speaker review checklist

For a native German speaker. Simple tick-box items. These are pure German
language facts that the project owner (not a German speaker) cannot verify.
None block the current VALIDATED_NOT_YET_PROVEN status; they refine confidence.

No front-matter (native-review file, exempt).

---

## [N-de-01] Adjective declension endings (strong/weak/mixed)

The full adjective-ending table (by article type × gender × case × number)
is morphological lexical data, not yet compiled. A native speaker should
confirm a compact reference table before adjective-agreement exercises ship.

- [ ] Table matches standard Duden declension.

## [N-de-02] Strong/irregular verb list

Strong verbs (vowel-changing stems: singen–sang–gesungen) and the irregular
sein/haben/werden need a verified conjugation table before verb exercises.

- [ ] Core strong-verb list (top ~100) verified.

## [N-de-03] Noun gender for common vocabulary

Gender (der/die/das) is not predictable and must be learned per noun. Confirm
the gender of the first lesson's vocabulary set once real content exists.

- [ ] Gender correct for lesson-1 nouns.

## [N-de-04] Naturalness / register of translations

Per ADR-016: confirm example sentences and translations sound natural to a
native speaker (not literal), and that du/Sie choice fits each situation.

- [ ] Sentences sound natural; du/Sie appropriate.

## [N-de-05] Two-way prepositions (Wechselpräpositionen) case usage

in/an/auf/über/unter/vor/hinter/neben/zwischen take Akkusativ (motion) or
Dativ (location). Confirm example sentences use the correct case.

- [ ] Case correct in examples.
