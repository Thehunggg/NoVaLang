# pt — Native speaker review checklist

For a native Brazilian Portuguese speaker. Simple tick-box items. Pure
Portuguese language facts the project owner (not a Portuguese speaker) cannot
verify. None block the current VALIDATED_NOT_YET_PROVEN status; they refine
confidence.

No front-matter (native-review file, exempt).

---

## [N-pt-01] Noun gender + irregular plurals

Gender (m/f) is largely lexical and `-ão` plurals split three ways
(coração→corações, pão→pães, mão→mãos). Confirm gender + plural for the first
lesson's vocabulary once real content exists.

- [ ] Gender + plural correct for lesson-1 nouns.

## [N-pt-02] ser vs estar boundary cases

The core rule (permanent vs temporary) is clear, but confirm borderline
examples (ser/estar with locations of events, with adjectives that change
meaning: é vivo / está vivo) sound natural.

- [ ] ser/estar choice correct in examples.

## [N-pt-03] Irregular verb tables

ser, estar, ter, ir, fazer, pôr, poder, querer are irregular. Confirm a compact
present-tense table (você/eu/nós/vocês baseline) before verb exercises ship.

- [ ] Core irregular-verb table verified (BR você baseline).

## [N-pt-04] Preposition + article contraction completeness

Confirm the contraction set (do/da/no/na/ao/à/pelo/pela + optional dum/deste/
nisso) is complete and that crase (à) usage in examples is correct.

- [ ] Contractions + crase correct.

## [N-pt-05] Clitic pronoun placement (BR colloquial vs written)

BR colloquial favors proclisis (Me diga) and subject-as-object (Eu vi ele);
written standard keeps o/a. Confirm example sentences match the intended
register (teach written, annotate colloquial).

- [ ] Clitic placement fits register.

## [N-pt-06] você vs tu regional appropriateness

Confirm that a você baseline reads as natural nationwide and that no example
forces tu conjugation. Flag any dialogue where a Northeastern/Southern speaker
would sound off with você.

- [ ] você baseline natural; no forced tu.

## [N-pt-07] Naturalness / register of translations

Per ADR-016: confirm example sentences sound natural to a Brazilian (not
literal), and você/o senhor choice fits each situation.

- [ ] Sentences natural; você/senhor appropriate.
