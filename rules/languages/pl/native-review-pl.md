# pl — Native speaker review checklist

For a native Polish speaker. Simple tick-box items. Pure Polish language facts
the project owner (not a Polish speaker) cannot verify. None block the current
VALIDATED_NOT_YET_PROVEN status; they refine confidence.

No front-matter (native-review file, exempt).

---

## [N-pl-01] Case ending tables (7 cases × 3 genders × number)

The full declension paradigm (including vocative and consonant/vowel
alternations: ręka→ręce, Bóg→Boże) is heavy morphological lexical data, not yet
compiled. Confirm a compact reference paradigm before case exercises ship.

- [ ] Declension tables match standard reference (e.g. Wielki słownik).

## [N-pl-02] Masculine-personal (virility) gender in plural

The męskoosobowy vs non-męskoosobowy split changes verb/adjective/numeral forms
in the plural. Confirm example sentences use the correct plural agreement.

- [ ] Virility agreement correct in plural examples.

## [N-pl-03] Verb aspect pairs

Which perfective matches which imperfective (robić/zrobić, pisać/napisać,
brać/wziąć) is largely lexical. Confirm the first lessons' aspect pairs.

- [ ] Aspect pairs correct for lesson vocabulary.

## [N-pl-04] Numeral–case agreement

Confirm example sentences apply the correct case after numerals (dwa stoły /
pięć stołów / dwóch mężczyzn / dwie kobiety), including the masculine-personal
numeral forms.

- [ ] Numeral case + virility correct.

## [N-pl-05] Noun gender + irregular declension

Gender is mostly predictable from ending but has exceptions. Confirm gender and
any irregular case forms for the first lesson's nouns.

- [ ] Gender + declension correct for lesson-1 nouns.

## [N-pl-06] ty / Pan / Pani appropriateness

Confirm ty vs Pan/Pani choice fits each situation and that formal address uses
3rd-person verb agreement correctly (Czy Pan mówi...?).

- [ ] Formal/informal address correct; 3rd-person verb with Pan/Pani.

## [N-pl-07] Naturalness / register of translations

Per ADR-016: confirm example sentences sound natural to a native (not literal),
aspect choice fits context, and word order matches information focus.

- [ ] Sentences natural; aspect + word order idiomatic.
