## Đặc trưng tiếng Indonesia nổi bật

- **Chữ Latin 26** + digraph ng/ny/sy/kh; **'e' nhập nhằng** [e]/[ə] (schwa).
- **KHÔNG biến tố** (không giống/cách/chia thì/số nhiều — điểm dễ nhất).
- **Thì qua hư từ** (sudah/sedang/akan).
- **PHỤ TỐ PHÁI SINH** (meN- biến âm mũi, ber-/di-/-kan/-i/ke-...-an) — cần học.
- **LÁY** (buku-buku, dấu nối); tính từ SAU danh từ.
- **Đại từ lịch sự** (saya/aku, Anda/kamu, kami/kita); Bapak/Ibu.
