# Croatian Grammar and Usage (hr-HR)

## 7 cách (ĐẶC TRƯNG SLAV — điểm khó nhất)

Danh từ + tính từ + đại từ + số đều biến theo **cách × giống × số**:

| Cách | Chức năng |
|---|---|
| nominativ | chủ ngữ |
| genitiv | sở hữu, phủ định, sau nhiều giới từ (iz Zagreba) |
| dativ | tân ngữ gián tiếp |
| akuzativ | tân ngữ trực tiếp |
| vokativ | hô gọi (Ivane!) |
| lokativ | nơi chốn (luôn sau giới từ: u školi) |
| instrumental | công cụ / cùng với (s prijateljem) |

**KHÔNG có mạo từ** — cách + trật tự thay chức năng xác định.

## Giống & số

3 giống (**muški** đực / **ženski** cái / **srednji** trung). Số ít / số nhiều
+ dạng đếm đặc biệt sau 2–4. Gợi ý giống: `-a` cái, phụ âm cuối đực, `-o/-e`
trung (nhiều ngoại lệ lexical).

## Động từ — chia ngôi + THỂ

- Chia **6 ngôi** (radim/radiš/radi/radimo/radite/rade).
- **THỂ (aspect)** perfective/imperfective là **cặp từ vựng** (pisati/napisati,
  čitati/pročitati) — đặc trưng Slav, khó.
- Quá khứ = **perfekt**: biti (enclitic sam/si/je) + phân từ `-l` hợp giống/số
  (radio/radila).
- Tương lai = **futur I**: htjeti (ću/ćeš/će) + infinitiv.

## Trật tự từ + enclitic vị trí 2

SVO nền nhưng **tương đối tự do** (cách đánh dấu chức năng). **Pro-drop** (ẩn
chủ ngữ). **Enclitic** (je, sam, se, ga, mu, ću…) đứng **vị trí thứ HAI** (luật
Wackernagel): *Vidio **sam** ga* / *Ja **sam** ga vidio*.
